# 🎓 RYM Darija Algérien

**Apprenez le Darija Algérien avec plus de 1000 mots !**

Une application PWA moderne pour maîtriser le dialecte algérien à travers des exercices interactifs.

![RYM App](icon.png)

## ✨ Fonctionnalités

### 📚 Vocabulaire Riche
- **1000+ mots** organisés en 23 thèmes
- **Chiffres** : de 1 à 5000+
- **6 catégories principales** :
  - Fondamentaux (Salutations, Chiffres, Famille, Logement, Nourriture, Vêtements...)
  - Grammaire (Articles, Adjectifs, Pronoms)
  - Vocabulaire (Santé, Travail, Études)
  - Compétences (Prononciation, Conversation, Présentation)
  - Culture (Fêtes, Coutumes, Histoire, Musique & Arts)

### 🎮 Modes d'Apprentissage

1. **📚 Vocabulaire** - Fiches interactives avec traductions
2. **🎯 QCM** - Questions à choix multiples avec correction immédiate
3. **🎤 Audio** - Écoute et enregistrement de prononciation
4. **🤖 IA Coach** - Questions aléatoires du coach IA

### 💾 Fonctionnalités Avancées

- **Profil utilisateur** - Sauvegarde nom, email, pays, bio
- **Progression** - Suivi des XP, niveaux, streak (jours consécutifs)
- **LocalStorage** - Données sauvegardées automatiquement
- **Boutons d'action** :
  - 📥 Charger - Récupère le profil sauvegardé
  - ⬇️ Installer - Ajoute l'app à l'écran d'accueil
  - 💾 Sauvegarder - Exporte le profil

### 📱 PWA (Progressive Web App)

- ✅ Installable sur mobile (iOS & Android)
- ✅ Fonctionne hors ligne (après première visite)
- ✅ Interface native avec icône personnalisée
- ✅ Responsive design

## 🎯 Système de Progression

- **XP** : +50 pour QCM complet, +25 par bonne réponse IA
- **Niveaux** : 1 niveau tous les 500 XP
- **Streak** : Suivi des jours consécutifs d'apprentissage
- **Profil** : Statistiques détaillées et badges

## 💡 Comment Utiliser

### 1️⃣ Apprendre le Vocabulaire
- Clique sur une catégorie → choisir un thème
- Onglet "📚 Vocab" pour voir tous les mots

### 2️⃣ Tester avec le QCM
- Onglet "🎮 QCM" pour des questions à choix multiples
- Gagne des XP à chaque bonne réponse

### 3️⃣ Pratiquer la Prononciation
- Onglet "🎤 Audio" pour écouter et enregistrer
- Utilise le 🔊 pour entendre la prononciation

### 4️⃣ Chatbot IA
- Clique sur "🤖 IA" dans le slider
- Le coach pose des questions aléatoires
- Réponds en Darija ou en Français

### 5️⃣ Gérer le Profil
- Clique sur 👤 en haut à gauche
- Remplis tes infos (nom, email, pays, bio)
- Sauvegarder/Charger tes données

## 🔧 Installation sur Mobile

### Android
1. Ouvre l'app dans Chrome/Edge
2. Appuie sur "⬇️ Installer" en bas
3. Confirme l'installation
4. L'app s'ajoute à l'écran d'accueil

### iOS (Safari)
1. Ouvre en Safari
2. Appuie sur "Partager" → "Sur l'écran d'accueil"
3. Ajoute l'app
4. Elle apparaîtra comme une app native
